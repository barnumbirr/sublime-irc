IRC Syntax
==========

Thank you for installing IRC Syntax for Sublime Text!

Provides syntax highlighting for IRC log files from:
  - irssi, weechat, HexChat, mIRC, ZNC, and Snak

IRC log files are auto-detected when possible. If not, select 'IRC'
from View > Syntax.

If you have any questions, comments, concerns, bugs, or patches, please open an
issue or pull request over on GitHub (https://github.com/barnumbirr/sublime-irc).
